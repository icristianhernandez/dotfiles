## Summary

Personal NixOS + Home Manager monorepo for multiple systems.

NixOS and Home Manager for a declarative system, using a custom (small) role-based guard for modules.

Fish shell with Starship in Kitty; Neovim for full sotfware dev env;

Various DE configured (GNOME currently principal, default)

Hosts for wsl and linux desktop by now.

## Quick start

```bash
# Rebuild a host
sudo nixos-rebuild switch --flake .#desktopthinkpad

# Run CI locally
nix run .#ci

```
